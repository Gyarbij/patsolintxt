# patsolintxt

Is a Discord bot powered by Azure OpenAI. The bot leverages the Google Custom Search Engine to fetch and summarize search results, providing concise and relevant responses.

## Features

- **Search**: Utilizes the Google Custom Search API to search the web based on user queries.
- **Summarize**: Uses OpenAI's GPT-4 model to summarize the content of search results.
- **Chat**: Can engage in conversation by using the GPT-4 model to generate responses.

## Prerequisites

You'll need the following:

- Python 3.8 or later
- Azure OpenAI API key
- Google Custom Search JSON API key and Search Engine ID
- Discord bot token

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Gyarbij/patsolintxt.git
cd patsolintxt
```

2. Install the required packages:

```pip install -r requirements.txt
```
